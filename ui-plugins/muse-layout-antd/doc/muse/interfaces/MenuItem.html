Optional icon

icon?: ReactNode

The icon to display on the item. If it's a string, it will be the icon name in antd icons. If the name is in kebab case, it will be auto add '-outlined' post fix and then convert to antd icon name in pascal case. e.g: 'file', 'question-circle'. They will be converted to FileOutlined, QuestionCircleOutlined. If it's already in pascal case, it will be used directly, e.g: 'FileOutlined'.
